# SaathiAI Design System

## Typography
- **Primary Font:** `Inter`, sans-serif
- **Weights:** 300, 400, 500, 600, 700

## Icons
- **Library:** Material Symbols Outlined

## Color Palette (Global)
- **Primary / Brand:** `#10b981` (Emerald 500)
- **Accent 1:** `#3b82f6` (Blue 500)
- **Accent 2:** `#f59e0b` (Amber 500)
- **Accent 3:** `#ef4444` (Red 500)

## Theme Variants
### Dark Theme (Surfaces 3, 4, Index)
- **Background:** `#0f172a` (Slate 900) or `#0b0f19`
- **Surface:** `#1e293b` (Slate 800) or `#111827`
- **Text:** `#f8fafc` (Slate 50) and `#94a3b8` (Slate 400)

### Light Theme (Surface 2)
- **Background:** `#ffffff`
- **Surface:** `#f8fafc`
- **Text:** `#0f172a` (Slate 900) and `#64748b` (Slate 500)

### WhatsApp Theme (Surface 1)
- **Background:** `#efeae2`
- **Primary Action:** `#00a884`

## UI Elements
- **Cards:** Rounded corners, subtle borders.
- **Scrollbars:** Slim, custom styled for webkit.
